### 프롬프트:
{레이아웃 유형} 기반의 HTML/CSS 구조를 검증해주세요.

분석 요청 사항:

1. HTML 구조 검증
- HTML5 시맨틱 요소 사용 적절성
- W3C 마크업 유효성 검사
- 접근성 트리 구조 분석
- 메타데이터 완성도

2. 반응형 디자인
- 모바일 퍼스트 접근 확인
- {브레이크포인트} 기준 레이아웃 동작
- 이미지/미디어 반응형 처리
- 유동적 타이포그래피 적용

3. CSS 아키텍처
- 선택자 특이성 관리
- CSS 변수 활용도
- 재사용 가능한 유틸리티 클래스
- 네이밍 컨벤션 일관성

4. 성능 최적화
- 중복 스타일 정의
- 미디어 쿼리 최적화
- CSS 번들 크기
- 애니메이션 성능

5. 크로스 브라우징
- {지원브라우저} 호환성 검증
- 폴백 스타일 구현
- 프리픽스 사용 현황
