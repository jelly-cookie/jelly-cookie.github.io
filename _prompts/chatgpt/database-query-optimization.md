### 프롬프트:
{DB유형} 데이터베이스의 쿼리 성능을 최적화해주세요.

분석 요청 사항:

1. 쿼리 실행 계획
- EXPLAIN 분석
- 인덱스 사용 현황
- 테이블 스캔 패턴
- 조인 최적화 상태

2. 인덱싱 전략
- 인덱스 설계 검토
- 복합 인덱스 구성
- 인덱스 크기 분석
- 인덱스 유지보수 계획

3. 성능 병목
- 느린 쿼리 로그 분석
- 메모리 사용량
- 디스크 I/O 패턴
- 동시성 처리 상태

4. 데이터 모델링
- 정규화 수준 평가
- 파티셔닝 전략
- 샤딩 필요성 검토
- 데이터 분산 상태

5. 운영 최적화
- 백업/복구 전략
- 모니터링 설정
- 유지보수 계획
- 스케일링 전략
